Perl History

Timeline

  • Version 1.0 of Perl was introducted by Larry Wall in 1987
  • Version 2.x, 4.x and 5.x were introduced over the years
  • Latest released is 5.36.1
  • The language is still under active developed

Perl Info

  • Perl is open-source and free for both personal and commercial use
  • Supported on various operating system including Windows, Linux and Mac OS X
  • Version 2.x, 4.x and 5.x were introduced over the years
  • Latest released is 5.36.1
  • Support for standard programming constructs including
    1. Conditional
    2. Looping
    3. Arrays and hash
    4. File IO
  • Support for more advanced programming feature
    1. Regular Expression
    2. Object Orientated Programming
  • Rich collection of user supported modules CPAN
  • Python's support for ML and AI, has contributed to reduced users of Perl

Where To Get Perl

  • Microsoft Windows
    1. ActiveState
    2. Strawberry Perl
  • Linux
    1. With Ubuntu Linux terminal, issue "sudo apt-get install perl" command
    2. Build from source code with "make, make test and make install" command
